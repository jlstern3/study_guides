GitHub Commands Study Guide

Click on the steps to reveal the corresponding git commands for pushing a repo to GitHub.

  1. Create local directory that will house your GitHub project
  2. Navigate into that directory and initialize a new Git repository
  3. Create a new file within project folder
  4. Stage new file
  5. Check status of changes
  6. Commit changes to local git repository (with a message included)
  7. Go to GitHub and create new remote repository then link local and remote repos
  8. Switch 'master' branch to 'main' branch
  9. Push your local repo to your remote repo on GitHub
